# aula-27-02-manual
### Manual do aluno

1- Áreas escolares: acho importante apresentar as áreas que poderão ser frenquentadas pelos alunos e estarão ali para auxulia-los quando necessário para que a experiencia seja mais parzerosa e divertida;

2- Respeito e convivência: importante pois é o principio de tudo e sem isso a convivencia chega a ser cansativa e improdutiva;

3- Regras e organização: explica os métodos que o aluno poderá ser avaliado e os horários a serem seguidos:

4- Saúde e segurança: deixa os aluno mais calmos por saber que tem auxilio a saúde caso necessário;

5- Equipe: para que os alunos se familiarizem com os profissionais da instituição.
